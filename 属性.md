基本/常见字段

title：字符串。文章标题。例：title: "PPT 演示入口"
date：日期/时间。文章发布日期。例：date: 2025-11-19T17:00:00+08:00
draft：布尔值。是否为草稿（true 则默认不发布除非 buildDrafts 打开）。例：draft: false
summary / description：字符串。文章摘要 / 描述（模板用于 meta description 或 social）。例：description: "直达 Vue PPT 演示页"
tags：字符串数组。文章标签（模板把它映射为 keywords、tag 列表等）。例：tags: ["Opinion"]
categories：字符串或数组。分类（主题可能映射到 taxonomy）。例：categories: ["Tech"]
aliases：URL 字符串或数组。用于旧链接重定向到新链接。例：aliases: ["/old-url/"]
slug / url：自定义页面 slug 或 URL，用于定制 permalink。
keywords：字符串数组。用于 meta keywords / schema。例：keywords: ["AIGC","Art"]
作者/时间类

author：字符串或数组。文章作者（模板可回退到 site.Param 的 author）。例：author: "Man in the Mirror"
lastmod：最后修改时间（Hugo 支持），用于排序/显示。
publishDate：可用于控制发布时间（如果需要）。
SEO / canonical / index 控制

canonicalURL：字符串。文章原始/规范 URL（用于 canonical 链接）。例：canonicalURL: "https://external.example.com/post"
robotsNoIndex：布尔值。若为 true，模板会避免在 production 环境下生成（判断示例见 head.html）。
hideSummary / hideAuthor：模板中检查的参数名（用于隐藏摘要或作者显示）。
ShowCanonicalLink：布尔（模板检查 .Params.ShowCanonicalLink）。
社交卡 / 封面 / 图片

cover：对象，常见子字段（主题大量使用）：
cover.image：字符串。封面图片路径或 URL。例：cover.image: "images/cover.png"
cover.alt：字符串。图片 alt 文本。
cover.caption：字符串。图说。
cover.relative：布尔。指明封面路径是否相对页面（模板据此选择 absURL 或 路径拼接）。
cover.responsiveImages：布尔。是否生成响应式图片。
images：数组。文章相关图片（模板中的 get-page-images 会读取 .Params.images，可回退到 site params images）。
audio：字符串或数组。OpenGraph 音频链接（模板会输出 og:audio）。
videos：字符串或数组。OpenGraph 视频链接。
展示/交互开关（页面级）

disableShare：布尔。若为 true 则隐藏分享按钮（模板判断 ne .Params.disableShare true）。
editPost：对象，模板用于生成“编辑”链接：
editPost.URL：基础编辑 URL（可在 site.Params 中设置默认）
editPost.Text：显示文字
editPost.appendFilePath：布尔，决定是否将文件路径追加到 URL
placeholder：字符串。用于搜索/输入框占位（模板读取 .Params.placeholder）
hideSummary：布尔。用于在列表页隐藏摘要（模板使用 .Param "hideSummary"）
结构/渲染相关

type / layout：Hugo 的内容类型或自定义 layout。
resources：页面资源（Hugo page bundle 相关），在模板中以 .Resources 使用而非 front-matter，但可通过 cover.image 指向 bundle 内文件名。
其他可用但不一定常见的自定义字段

keywords（已列）— 用于 schema/JSON-LD。
canonicalURL（已列）、placeholder（已列）。
任意自定义 Params.* 字段：主题通过 .Params.<name> 访问页面级自定义参数（例如 .Params.customField），所以你可以按需添加字段，模板只会在显式使用该字段的情况下读取显示。
说明与实践建议

仓库根 default.md（见仓库）给出的示例只有 date、draft、title、tags，但主题 PaperMod 模板实际引用了许多 page-level params（如 cover、images、canonicalURL、keywords、author、editPost、disableShare、robotsNoIndex 等），因此写文章时可以安全使用上面的字段。
若想确保某个字段被模板使用或以特定方式渲染，请检查主题中对应模板（例如 cover.html、head.html、opengraph.html 等），我已经扫描了这些模板并把常用字段列在上面。
示例 front-matter（YAML 或 TOML 或 Hugo 的 +++ TOML 语法）与类型要匹配（数组要用 []，对象用嵌套语法），可参考仓库中现有文章的写法（content/posts/*
